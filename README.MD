Small Project to give a Player some Advantages in Among Us, if used make sure you have all agreeing to you using it.

### How to Build
1. Create a DLL Project in C#
2. Include all Files from Source
3. Install / Get BepInEx and Link against any DLL in your Among Us Folder
4. Hit Build in Release

### How to use
1. Get BepInEx
2. Place any folder from game_folder into your Among Us folder
3. Place the produced DLL in the Plugins Folder of BepInEx
